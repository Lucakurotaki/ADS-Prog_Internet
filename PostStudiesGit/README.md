# PostStudies

## Descrição

Projeto desenvolvido para estudos de backend na disciplina Programação para Internet II, e consiste em criação de um sistema de posts.

## Funcionalidades

 - Autenticação
 - Postagem
 - Sistema de follow
 - Curtir

## Tecnologias

 - Typescript
 - Node.js/Express
 - PostgreSQL
 - Docker
